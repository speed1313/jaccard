# Jaccard Similarity

A simple web app to compute [Jaccard similarity](https://en.wikipedia.org/wiki/Jaccard_index) between two texts by tokenizing them using OpenAI's [tiktoken](https://github.com/openai/tiktoken).

## Test

```bash
npm install
npm run start
```

```bash
npm run build

```
